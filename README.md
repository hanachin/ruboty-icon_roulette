# Ruboty::IconRoulette

ruboty handler for change bot's icon at random.

## Installation

Add this line to your application's Gemfile:

    gem 'ruboty-icon_roulette'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-icon_roulette

## Usage

```
> ruboty icon roulette!
```
